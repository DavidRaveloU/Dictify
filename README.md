##  Overview

Dictify is a sleek, offline-friendly dictionary app designed for word enthusiasts, students, and writers. It delivers instant definitions, pronunciation audio, grammatical breakdowns (noun/verb/adjective), and usage examples‚Äîall in a lightning-fast interface. Built with React Native and Expo for cross-platform performance
